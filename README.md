# QuickLook.Plugin.DumpBin
此插件用来查看dll、lib库的导出信息.
<img width="1002" height="834" alt="image" src="https://github.com/user-attachments/assets/77cedd42-422c-41b5-9098-f414d324f4c4" />


## Try out

1. Go to [Release page](https://github.com/QL-Win/QuickLook.Plugin.DumpBin/releases) and download the latest version.
2. Make sure that you have QuickLook running in the background. Go to your Download folder, and press <key>Spacebar</key> on the downloaded `.qlplugin` file.
3. Click the “Install” button in the popup window.
4. Restart QuickLook.
5. Select .dll file and press <key>Spacebar</key>.

## 设置
右键托盘图标->打开数据文件夹->编辑配置文件QuickLook.Plugin.DumpBin.config
``` xml
<Settings>
  <width>1000</width>
  <heigh>800</heigh>
</Settings>
```
